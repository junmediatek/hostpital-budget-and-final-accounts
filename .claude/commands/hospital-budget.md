# /hospital-budget

补全医院名单 Excel 中所有医院的 2024 决算、2025 预算、2026 预算总收入。

## 输入

- 不需要命令参数。
- 固定读取工作区根目录下的工作簿：`公立三级&二级医院名单.xlsx`。
- 表头在第 1 行。
- A 列：序号。
- B-D 列：地区信息。
- E 列：医院名称。
- F-H 列：需要写入的收入字段。
- I 列：来源链接。
- J 列：必要备注。

## 执行

1. 使用 xlsx skill 读取工作簿，找出所有医院行。
2. 按 Excel 行号从上到下为医院创建 `hospital-budget-final-accounts` Subagent 任务。
3. 任意时刻最多只能有 3 个 Subagent 正在执行；必须等已有 Subagent 完成后，才能补充创建新的 Subagent。
4. 传给每个 Subagent：固定工作簿路径、行号、A 列序号、B-D 地区信息、E 列医院名称、`resources` 路径。
5. Subagent 返回后，先记录结构化结果、缺失项、备注和已保存资料路径。
6. 每累计完成 3 个 Subagent，就使用 xlsx skill 将这 3 个结果按行号写回同一工作簿的 F、G、H、I、J 列；不要等所有
Subagent 全部完成才写回。
7. 所有 Subagent 完成后，如剩余未写回结果不足 3 个，也必须写回同一工作簿的 F、G、H、I、J 列。
8. 每次写回完成后保存工作簿，并重新读取本批次已更新行验证 F、G、H、I、J 列已保存正确。
9. 全部验证完成后，汇总输出处理结果。

## 规则

- 每个 Subagent 仍然只处理一家医院，不得跨医院批量处理。
- 一次最多并发执行 3 个 Subagent，不得超过该上限。
- 每完成 3 个 Subagent 写回一次 Excel；最后不足 3 个的剩余结果在所有 Subagent 完成后立即写回。
- 写回 Excel 时必须按 Subagent 返回的行号匹配原始医院行，不得依赖返回顺序。

## 写回格式

- F：`2024年决算-总收入`
- G：`2025年预算-总收入`
- H：`2026年预算-总收入`
- I：`2024决算: URL; 2025预算: URL; 2026预算: URL`
- J：只写必要备注；无备注则留空。

## 输出

简短说明已处理医院数、成功更新数、缺失项、工作簿路径和资料保存路径。
