# /hospital-budget

补全医院名单 Excel 中所有医院的 2024 决算、2025 预算、2026 预算总收入。

## 输入

- 不需要命令参数。
- 固定读取工作区根目录下的工作簿：`公立三级_二级医院名单.xlsx`。
- 来源附件保存目录：工作簿同级的 `resources/` 文件夹。
- 表头在第 1 行，数据从第 2 行开始；传给 Subagent 的行号为 Excel 实际行号（从 2 起）。
- A 列：序号。
- B-D 列：地区信息。
- E 列：医院名称。
- F-H 列：需要写入的收入字段。
- I 列：来源链接。
- J 列：必要备注。

## 执行

1. 使用 xlsx skill 读取工作簿，找出所有医院行（数据行，跳过表头）。
2. 按 Excel 行号从上到下，每批同时创建 3 个 `hospital-budget-final-accounts` Subagent 任务；必须等这 3 个全部完成、结果写回 Excel 后，才能创建下一批 3 个。
3. 传给每个 Subagent：工作簿绝对路径、Excel 行号、A 列序号、B-D 地区信息、E 列医院名称、`resources/` 绝对路径。
4. 本批 3 个 Subagent 全部返回后，记录每个的结构化结果（行号、三年收入、来源 URL、备注、附件路径、缺失项）。
5. 使用 xlsx skill 按行号将本批结果写回工作簿 F、G、H、I、J 列，写完后保存；写回完成后，再创建下一批 Subagent。
6. 所有批次处理完毕后，若最后一批不足 3 个，同样等该批全部返回后按相同流程写回并保存。
7. 全部写回完成后，汇总输出处理结果。

## 规则

- 每个 Subagent 只处理一家医院，不得跨医院批量处理。
- 每批同时创建 3 个 Subagent，必须等本批全部完成后才能创建下一批。
- 写回时必须按行号匹配医院行，不得依赖返回顺序。
- 某年数据缺失时，对应的 F/G/H 列写入空字符串，I 列该年 URL 写 `""`，J 列注明缺失原因。

## 写回格式

- F：`2024年决算-总收入`（数值，单位万元；缺失则留空）
- G：`2025年预算-总收入`（数值，单位万元；缺失则留空）
- H：`2026年预算-总收入`（数值，单位万元；缺失则留空）
- I：`2024决算: URL; 2025预算: URL; 2026预算: URL`（无 URL 则该项写 `""`）
- J：只写必要备注（缺失原因、数据说明等）；无备注则留空。

## 输出

处理完成后，以表格形式输出汇总：

| 项目 | 内容 |
|------|------|
| 已处理医院数 | N |
| 成功更新数 | N |
| 数据缺失项 | 列出医院名及缺失年份 |
| 写回失败行 | 列出行号及原因（如有） |
| 工作簿路径 | 绝对路径 |
| 附件保存路径 | `resources/` 绝对路径 |
